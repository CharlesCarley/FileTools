Detailed Description

Base class for a file in this system. Derived classes should override the table methods and supply them to this class.

Parameters

Returns

◆ setFilterList()

void ftFile::setFilterList ( SKhash *  filter,
SKuint32  length,
bool  inclusive = false 
)

This enables a filter for structures.

Parameters
filterThe input filter.
lengthThe number of structures in the filter
inclusiveSets whether the filter should include or exclude the supplied filter.

Internally struct types are stored by a hash of their unique type name. The filter list needs to be a null terminated SKhash [] array.

The hash function resides in the ftCharHashKey class Usage:

FBThash filterList[] = {
ftCharHashKey("StructToFilter1").hash(),
ftCharHashKey("StructToFilter2").hash(),
nullptr
};
