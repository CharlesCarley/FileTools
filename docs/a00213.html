ftToken Class Reference

Detailed Description

Is a temporary data storage class to store a c/c++ language semantic. It is used when extracting member data with ftScanner.

int ftToken::getArrayLen ( ) const
inline

Gets the defined array size of c/c++ data type.

If the type is not defined as an array in c/c++, the array length is defined to 1. Otherwise, the value that is returned should be less than or equal to the FT_ARRAY_DIM_MAX value.
