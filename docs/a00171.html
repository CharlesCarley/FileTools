ftScanDna Class Reference

Detailed Description

ftScanDna is a utility class to read minimal information from the file by skipping from chunk to chunk.

The idea of the DNA scan is to jump to the DNA1 block first, extract the table data, then seek back to the file header offset and read the chunks up to the DNA1 block. Then, create associations with the structure and member declarations along with every chunk.
