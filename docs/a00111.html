ftChunkScan Struct Reference

Detailed Description

ftChunkScan is a structure that contains the minimum amount of information to jump around the file.

This is used when scanning for a specific data block. The code is the specific chunk id that is being searched for. The length is used to seek over the current chunk's data so that the next chunk can be found.
