◆ compare()

bool ftMember::compare ( ftMember *  rhs) const

Used to test semi-equal members so that an offset to the correct member in both file and memory buffers can be matched.

Parameters
rhsThe other member to test against. Regardless of what 'this' points to (fileMember or memoryMember) the rhs parameter needs to be in the opposite table.
Returns
True if the internal keys match.

The data may change over time:

int foo; => short foo;
int bar[3]; => int bar[8];

Ideally this is meant to weed out those subtle changes. In the current state, this is not 100 percent.
