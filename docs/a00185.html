Member Enumeration Documentation

◆ Flag

enum ftStruct::Flag
Enumerator
CAN_LINK 
MISSING 

Flag that indicates that this structure was not found in either the file or memory table.

MISALIGNED 

Flag that indicates a misalignment between loaded and calculated offsets to member data. If this is reported the loader cannot jump directly to the member after the first misaligned offset. Which means that something is corrupted.

NEED_CAST 

This flag is used to determine whether or not the file an memory versions of this structure are in sync. If they are out of sync, then each member in the file structure needs to be cast into each member of the memory structure.

HAS_DEPENDENT 

Internal flag. If this is set, then this structure has other structures for members.

◆ find()

ftMember * ftStruct::find ( ftMember *  other) const

Preforms a non recursive binary search for the supplied member.

Parameters
otherThe corresponding member to find.
Returns
Null if other is null, other's search key is not defined, or if other is not found. Otherwise it will return the found corresponding member.

This method is meant to be used to find a previously declared member in another table. The member may have changed slightly between now and the time it was originally saved. It is not guaranteed that this will return the exact member in question. The accuracy is dependent on how well the search key is built.

bool ftStruct::hasDependentTypes ( ) const
