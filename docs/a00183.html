SKuint32 ftType::id

Contains the structure index if the type is a structure. Otherwise, this field is unused and should be set to data type limit for an unsigned int.

Caution must be used here. All data types are stored in a single array of structures. This is the index into that array with an offset of the total number of atomic types.
